Why are all metrics 0.000?

The framework is running in stub mode because no ANTHROPIC_API_KEY is set. In stub mode, every LLM call returns a deterministic hash string like [stub:claude-haiku-4-5:abc123] Answer for input len 42. — the judge scores these as 0 because they contain no real answer content.

This is actually useful — it proves the full pipeline works end-to-end without spending any money. Once you add an API key, the same pipeline produces real scores.

To get real results — do ONE of these:

Option A (Docker): Stop the container, edit .env at repo root, set ANTHROPIC_API_KEY=sk-ant-api03-..., then run docker compose up --build

Option B (shell): $env:ANTHROPIC_API_KEY = "sk-ant-..." (PowerShell) or export ANTHROPIC_API_KEY="sk-ant-..." (bash), then restart uvicorn

Which model to use: Type your model id in the Upload page's Model field (e.g. claude-sonnet-4-6 or claude-haiku-4-5). Haiku is cheapest for experiments.
